Arquivo auxiliar funcionamento do projeto:

1 - A classe "Main_Lexer" testa o léxico.
2 - A classe "MyLexer" é uma herança de Lexer e resolve o problema dos parentêses aninhados/balanceados.
3 - A clase "Main_Parser" testa o sintático.
5 - A gramática se encontra dentro da pasta "src" do projeto com o nome "portugol.grammar".
6 - O arquivo de teste usado por mim encontra - se na pasta principal do projeto com o nome "teste.txt".
7 -Para testar diretamente no eclipse usar linhas comentadas logo no inicio do codigo ou adicionar o 
caminho do .txt teste nos argumentos da execucao
8 - Dentro da pasta principal do projeto também encontra - se um .jar, "execute_prompt", para realizar os
teste via prompt de comando. (Atenção ao gerar novo .jar, definir a classe principal, nesse caso já está
com o Main_Parser).
9 - Através do prompt de comando, podemos passar o argumento diretamente via a linha de comando de execução
ou mandar rodar somente e digitar o caminho.
